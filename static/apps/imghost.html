ImgHost

An Electron app of imghost


imghost是一个使用electron开发的跨平台图床应用

简介

该应用基于electron，你可以使用它自由上传图片到定义的图床api或者批量下载图片。使用json文件保存配置

下载

使用git克隆源码仓库

$ git clone https://github.com/Landers1037/imghost.git

编译安装文件

$ cd imghost && npm install
$ npm run build
