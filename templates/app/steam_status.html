Steam Status

Steam friends status watchdog


Steam status是一个steam好友在线状态监控爬虫工具，同时是一款CLI工具

简介

你可以使用CLI工具在本地测试抓取好友的在线状态，或者使用服务端工具部署于服务器。该工具会默认以5分钟/次的频率爬取在线状态。
注意，steamcommunity的ip不稳定，在你使用普通网络环境时可能会爬取失败。
该工具服务端使用redis保存你爬取的数据，你可以在main.py文件中关闭该选项

使用示例


                git clone https://github.com/Landers1037/steam_status.git

                #按照用户名模糊匹配
                steam_status.py run --name=用户名
                #精准抓取 返回状态信息
                steam_status.py run --url=用户主页地址

                #使用帮助
            Usage: steam_cli.py [OPTIONS] COMMAND [ARGS]...

            Options:
              --help  Show this message and exit.

            Commands:
              run      指定用户名称或准确的用户主页地址
              version  版本信息
